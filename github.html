Guia Git e GitHub

O que é o Git

Git é um sistema de controle de versão distribuído, usado para rastrear mudanças em arquivos e coordenar trabalho entre programadores.

O que é o GitHub

GitHub é uma plataforma online que hospeda repositórios Git, permitindo colaboração, versionamento e gerenciamento de projetos.

Diferenças entre Git e GitHub

  • Git: Ferramenta local de controle de versão.
  • GitHub: Serviço online para hospedar repositórios Git.

Principais comandos Linux para usar no Git Bash

  • ls – lista arquivos
  • cd – muda de diretório
  • pwd – mostra o caminho atual
  • mkdir – cria diretório
  • touch – cria arquivo
  • rm – remove arquivo
  • cat – mostra conteúdo de arquivo

Comandos do README do repositório

  • git add . – adiciona arquivos
  • git commit -m "mensagem" – salva alterações
  • git push – envia para o GitHub
  • git clone – clona repositório

Como criar um repositório no GitHub

Vá até o GitHub, clique em "New", preencha nome e descrição, e clique em "Create repository".

Como ligar a um projeto existente na máquina

Use git remote add origin URL para conectar ao repositório remoto.

Como gerar uma versão com Git

Use git commit -m "mensagem" para salvar uma versão do projeto.

Como enviar da máquina para o GitHub

Use git push origin main para enviar os arquivos para o repositório remoto.
